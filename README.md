# BeagleBone Black Linux Bring-up

A Roll Your Own (RYO) Embedded Linux project for the BeagleBone Black.

This repository documents my journey building a custom Linux system for the
BeagleBone Black, including:

- Cross-compilation toolchain using crosstool-NG
- U-Boot bootloader build
- Linux kernel compilation
- BusyBox root filesystem creation
